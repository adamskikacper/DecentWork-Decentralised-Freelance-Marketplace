# Button Component

The Button component is a versatile UI element used to trigger actions when clicked.
It supports various styles, sizes, and can include icons.

## Usage

```tsx
import { Button } from "@/components/UI/button";

// Basic usage
<Button>Click me</Button>

// With variants
<Button variant="destructive">Delete</Button>
<Button variant="outline">Cancel</Button>

// With sizes
<Button size="sm">Small</Button>
<Button size="lg">Large</Button>

// With icons
<Button>
  <Heart />
  Like
</Button>
```
