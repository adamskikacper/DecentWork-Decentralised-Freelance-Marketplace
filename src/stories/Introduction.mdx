# Decentralized Freelance Marketplace Component Library

Welcome to the component library for the Decentralized Freelance Marketplace! This Storybook provides an interactive documentation of all UI components used in the application.

## What's included

- **UI Components**: Basic building blocks like buttons, inputs, cards
- **Form Components**: Complex form elements and validation patterns
- **Layout Components**: Page layouts, navbars, and dashboard elements
- **Dashboard Components**: Specialized components for freelancers and clients

## How to use this Storybook

1. **Browse components**: Use the sidebar to navigate through component categories
2. **View examples**: Each component comes with various usage examples
3. **Interact with props**: Adjust component props in the Controls panel
4. **View documentation**: Check out the Docs tab for detailed API information
5. **View component code**: The Show Code button reveals the component's implementation

## Component architecture

Our components are built with:

- **React**: For component structure and state management
- **TypeScript**: For type safety
- **Tailwind CSS**: For styling
- **shadcn/ui**: For consistent UI design patterns
- **Radix UI**: For accessible UI primitives

## Accessibility

All components are designed with accessibility in mind:

- Proper semantic HTML
- ARIA attributes
- Keyboard navigation
- Color contrast
- Screen reader support

## Contributing

When adding new components:

1. Create the component in the appropriate directory
2. Add TypeScript types and props documentation
3. Create a story showing various states and variants
4. Include accessibility considerations
5. Add MDX documentation if needed
